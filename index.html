Overview

O Titan é um framework para instanciação de gerenciadores de conteúdo (do inglês, Content Management Systems - CMS), permitindo a criação de aplicações Web complexas. A proposta do Titan Framework é ser uma ferramenta de desenvolvimento simples e completa, permitindo criar aplicações robustas de forma rápida e fácil. Executa em servidores Web com PHP (5.3 ou superior) e com banco de dados PostgreSQL (8.4 ou superior).
